
Snake

Description:
This project is a simple Snake game implemented using Python and Pygame.
The player controls a snake that moves around the screen, eats food, grows longer, and increases score and level.

Features:

* Snake movement using arrow keys
* Wrap-around (snake appears on the opposite side of the screen)
* Random food generation (not on the snake)
* Score and level system
* Increasing speed with each level
* Game over when the snake collides with itself

Controls:

* Up arrow: move up
* Down arrow: move down
* Left arrow: move left
* Right arrow: move right

How it works:
The snake is represented as a list of coordinates.
Each move adds a new head and removes the tail.
When the snake eats food, it grows longer.
Every 3 foods increases the level and speed.
