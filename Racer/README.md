
Racer

Description:
This is a simple 2D car game built using Pygame. The player controls a car,
avoids enemy cars, and collects coins.

Features:
- Player movement (left and right)
- Random enemy spawning
- Coin collection system
- Score counter
- Increasing speed over time
- Game Over screen

Controls:
LEFT ARROW - move left
RIGHT ARROW - move right
ESC - exit game

How the Game Works:
- Player starts at the bottom of the screen
- Enemies spawn from the top and move down
- Coins also fall from the top
- Avoid enemies to survive
- Collect coins to increase coin count
- Each avoided enemy increases score
- Collision with enemy ends the game
